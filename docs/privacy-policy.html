Privacy Policy

Action Overlay for macOS — Last updated: March 2, 2026

Summary: Action Overlay does not collect, store, or transmit any personal data. All processing happens locally on your Mac.

Data Collection

Action Overlay does not collect any personal information. The app:

  • Does not require an account or sign-in
  • Does not use analytics or tracking
  • Does not contain advertising
  • Does not access your contacts, photos, calendar, or location
  • Does not use cookies or similar tracking technologies

Local Processing

All video processing, telemetry parsing, and overlay rendering happens entirely on your Mac. Your video files, telemetry data (SRT files, flight records), and exported videos never leave your device.

Network Access

Action Overlay makes a single type of network request:

  • DJI Flight Record Decryption: When you load a DJI flight record file, the app contacts the DJI API (dev.dji.com) to retrieve decryption keychains. Only the flight record's encryption metadata is sent — no personal data, video files, or GPS coordinates are transmitted. This request requires a DJI developer API key that you provide.

No other network requests are made. The app functions fully offline for all other features, including SRT telemetry processing and video export.

File Access

Action Overlay only accesses files that you explicitly select through macOS file dialogs (Open/Save panels). The app operates within the macOS App Sandbox and cannot access files outside of your selections.

Third-Party Services

Action Overlay does not integrate with any third-party analytics, advertising, or data collection services. The only external service used is the DJI developer API as described above.

Data Storage

The app stores your preferences (overlay settings, device presets, window positions) locally using macOS UserDefaults. This data stays on your Mac and is never transmitted.

Children's Privacy

Action Overlay does not knowingly collect any information from children under the age of 13.

Changes to This Policy

If this privacy policy is updated, changes will be posted on this page with an updated date. Since the app does not collect data, significant changes are unlikely.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub support page.
